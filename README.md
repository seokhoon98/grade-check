# 학기말 성적 산출 시스템

정기시험(중간·기말)과 수행평가 점수를 반영 비율에 따라 자동 계산하여 최종 성적을 산출하는 Streamlit 대시보드입니다.

## 주요 기능

- **반영 비율 설정**: 중간고사 / 기말고사 / 수행평가(최대 10항목) 비율 자유 설정
- **점수 입력**: 웹 테이블 직접 입력 또는 Excel 업로드
- **성적 자동 산출**
  - 환산점수 계산 (취득점수 / 만점 × 반영비율 × 100)
  - 석차, 성취도(A~E), 석차등급(1~9등급) 자동 부여
  - 결시 처리 지원
- **대시보드**: 점수 분포, 성취도 비율, 석차등급 분포, 영역별 평균 시각화
- **Excel 다운로드**: 최종 성적표 Excel 파일로 저장

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud 배포

1. 이 저장소를 GitHub에 push
2. [share.streamlit.io](https://share.streamlit.io) 접속
3. 저장소 연결 후 `app.py` 선택하여 배포
